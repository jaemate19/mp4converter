# MP4 Converter Web Application

## 🚀 Overview
This project is a web-based media conversion tool that allows users to upload video files and convert them using server-side processing.

It demonstrates backend development, file handling, and integration with multimedia processing tools.

---

## ⚙️ Features
- File upload functionality
- Server-side media processing
- Integration with FFmpeg
- Conversion of video/audio formats
- Backend validation and handling

---

## 🧠 How It Works
1. User uploads a media file
2. The server processes the file using FFmpeg
3. The converted file is generated
4. Output is returned to the user

---

## 🛠️ Tech Stack
- PHP
- FFmpeg
- HTML/CSS
- Apache
- Linux VPS

---

## 📦 Installation

```bash
git clone https://github.com/jaemate19/mp4converter.git
cd mp4converter
chmod +x setup.sh
./configure
```
